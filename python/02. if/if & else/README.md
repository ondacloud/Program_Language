# IF & ELSE
**IF & ELSE란?** <br>
조건이 참과 거짓일 때 실행되는 조건문 함수

<br>

## IF & ELSE Basic Structure
```python
if <Condition>:
    <Command>
    ...
else:
    <Command>
    ...
```

<br>

## Example
### IF & ELSE Sturecture
```python
a = int(input())

if (a < 0):
    print("successed")
else:
    print("failed")
```
