# IF & ELIF & ELSE
**IF & ELIF & ELSE?** <br>
조건이 참과 거짓일 때 실행되는 조건문 함수

<br>

## IF & ELIF & ELSE Basic Structure
```python
if <Condition>:
    <Command>
    ...
elif <Condition>:
    <Command>
    ...
else:
    <Command>
    ...
```

<br>

## Example
### IF & ELIF & ELSE Sturecture
```python
a = int(input())

if (a < 0):
    print("successed")
elif (a = 0):
    print("equal")
else:
    print("failed")
```
