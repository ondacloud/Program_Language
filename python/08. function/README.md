# Function
**Function이란?** <br>
특정 작업을 수행하고 결과를 반환하는 코드 블록

<br>

## Function Basic Structure
```python
def <Function Name>():
    <Command>
    ...
    return <Value>

<Function Name>()
```

<br>

## Example
### Function Structure
```python
def sum_num(a, b):
    return a+b

a = 1; b= 2
c = sum_num(a, b)
print(c)
```
