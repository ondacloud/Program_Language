# Print
**Print란?** <br>
텍스트 출력하는 함수

<br>

## Print Baic Structure
```python
print("<Text>")
```

<br>

## Output Funtion

|Format Specifier|Description|
|---|---|
|%d|10진수 정수형|
|%f|실수형|
|%c|문자|
|%s|문자열|

## Example
### Print Structure - 1
```python
print("Hello World!")
```

### Print Structure - 2
```python
print("%s" % "Hello World!")
```
