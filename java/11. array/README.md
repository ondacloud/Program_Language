# Array
**Array란?** <br>
여러 연속적인 값을 저장하고자 할 때 사용하는 자료형
> Array의 시작은 0부터 시작합니다.

<br>

## Array Baic Structure
```java
public class Main {
    public static void main(String[] args) {
        <Data Type>[] <Variable> = new <Data Type>[<Array Size>];
    }
}
```
